## Required packages

- Qt5 >= 5.15.3
- QtAV >= 1.13.0
- libmediainfo >= 22.03
- libzen >= 0.4.39
- pyUserInput (temporary)

## Shotcuts

- Ctrl+A: Open files.
- Ctrl+O: Open files.
- Alt+Enter: Fullscreen.

## Installation instructions

To install "OMPlayer", use the following commands:
```sh
$ git clone https://github.com/mxnt10/OMPlayer.git
$ cd OMPlayer

$ mkdir build
$ cd build; cmake ..
$ make
```
